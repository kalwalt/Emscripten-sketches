# Emscripten-sketches
Some examples to compile C++ code to WASM or javascript thanks to Emscripten.

## List of examples
Every example is located in a separate folder. For every example is provided a build script and precompiled binaries.

- ARLOG
- gl_textures
- hello_world_class
- Hello_world_sdl
- hello_world_worker
- libwebp-wasm
- node_file_loader
- pre-js-ES6-example
- struct-with-vectors
- test_write
- worker_api
- x2
